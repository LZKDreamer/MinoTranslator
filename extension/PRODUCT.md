# Product

## Register

product

## Users

**主要用户群：** 观看外语 YouTube 视频、需要实时字幕翻译的用户。
**次要用户群：** 日常浏览外文网页、需要快速划词翻译的用户。

**使用场景：** 
- 用户在 YouTube 全屏观看视频时，译文字幕需要清晰可读、不遮挡画面
- 用户在浏览器中浏览外文网页时，选中文字后希望立刻看到翻译
- 用户安装插件后希望"开箱即用"——只需填入 API Key 即可使用

## Product Purpose

一款 Chrome 翻译扩展，核心功能为 YouTube 视频字幕翻译，辅助功能为页面划词翻译。提供多种主流翻译模型支持（默认 Agnes AI），用户可根据需求自备 API Key 选择模型。

**成功标准：** 
- 用户安装后 2 分钟内完成配置并开始使用
- 字幕翻译与视频播放同步：当前区域译文最先出现，拖拽进度条自动重排优先级
- 切换视频不留残影，跨视频字幕不混淆
- 划词翻译响应迅速，不干扰正常浏览

## Brand Personality

**简洁 · 高效 · 友好**

- 界面干净克制，功能一目了然，不分散用户注意力
- 每个操作都有明确目的，不多余、不花哨
- 使用体验顺畅自然，像工具应有的样子——精准、可靠、不打扰

## Anti-references

- **不拥挤**：不用卡片堆叠、不用密集的信息层级。每个界面只展示当前需要的功能
- **不花哨**：不使用彩虹配色、毛玻璃、过度动画、渐变文字。色彩服务于功能，而非装饰
- **不过度设计**：不是"炫酷的 AI 工具"，而是一个"好用的翻译插件"

## Design Principles

1. **隐身好用** — 工具在需要时出现，不需要时消失。字幕翻译装好后，用户应该专注于视频内容，而非工具本身
2. **一事一屏** — Popup 只做高频操作（开关、模式切换），模型配置等低频操作放在 Options 页面。不做功能堆叠
3. **一次配置，持续使用** — 默认 Agnes AI 预填好地址和模型 ID，用户只需填 Key。换了模型也不影响已有工作流
4. **克制的视觉** — DeepL 式的蓝白极简风格为基础。色彩是引导信号，不是装饰。留白是功能，不是浪费

## Accessibility & Inclusion

- 基本 WCAG 标准：正文文本与背景对比度 ≥ 4.5:1，大号文本 ≥ 3:1
- 字幕字体大小可调，支持半透明背景保证可读性
- 设置页面文本清晰可缩放
